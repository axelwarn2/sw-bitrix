Шаги для решения [ex1-58] Создание уменьшенной картинки: фотография сотрудника

1. Перейти к шаблону комплексного компонента 
    1.1 Найти комплексный компонент, для которого необходимо доработать шаблон. Это может быть, например, список сотрудников. 
    1.2 Перейти в административную панель:
        Контент > Структура сайта > Найти страницу, где выводится компонент 
    1.3 Открыть шаблон компонента для редактирования: 
        - Перейти в папку шаблона сайта: /bitrix/templates/ваш_шаблон/components/название_компонента/ 
        - Найти файл template.php внутри шаблона компонента.

2. Добавить логику уменьшения изображения через API Bitrix 
    2.1 Открыть файл template.php для редактирования. 
    2.2 Найти участок кода, где выводится детальная картинка сотрудника. 
    2.3 Добавить код для уменьшения изображения с помощью функции CFile::ResizeImageGet, которая является частью API Bitrix.

    Пример кода для уменьшения изображения:

    <?php
    // Проверка наличия детальной картинки
    if (isset($arItem["PREVIEW_PICTURE"]["SRC"])) {
            $Img = CFile::ResizeImageGet($arItem["PREVIEW_PICTURE"], array('width' => 68, 'height' => 59), BX_RESIZE_IMAGE_PROPORTIONAL, true);
            $Img = $Img["src"];
        }
        <?php
    }
    ?>

3. Проверить изменения на сайте 
    3.1 Сохранить файл template.php. 
    3.2 Открыть страницу с компонентом на сайте и проверить, что фотографии сотрудников выводятся с размерами 68x50 пикселей. 
    3.3 Убедиться, что изображение действительно уменьшается на back-end через API Bitrix, а не с помощью браузера (проверить можно через просмотр исходного HTML-кода страницы).

4. Тестирование работы 
    4.1 Добавить новую фотографию сотрудника с высоким разрешением и убедиться, что она корректно уменьшается до нужных размеров при выводе на странице. 
    4.2 Проверить отображение на разных устройствах и разрешениях экрана.
