Шаги для решения [ex1-13]:
1: Импортировать информационный блок Отзывы из xml файла, файл доступен в материалах
1.1) Администрирование > Контент > Инфоблоки > Импорт > XML
1.2) Выбрать файл rew.xml, который находится в папке demo-content-iblock для загрузки
1.3) Выбрать тип инфоблока "Товары и услуги"
1.4) Убедиться, что активирована опция "Генерация символьного кода транслитом" для новых элементов
(Типы инфоблоков > Товары и услуги > Отзывы > Поля > Символбный код)

2: В списке элементов должны присутствовать только такие поля: ID, Активность, Начало активности, Название, Должность, Компания
2.1) Перейти в список элементов инфоблока: Администрирование > Контент > Инфоблоки > Список элементов
2.2) Настроить видимые поля для отображения: ID Активность Начало активности Название Должность Компания
     Для этого в списке элементов выбрать кнопку "Настроить список" и отметить необходимые поля

3: В фильтре должен быть доступен поиск по полю Должность
3.1) В разделе Контент > Инфоблоки > Список элементов, открыть панель фильтрации
3.2) Нажать "Настроить фильтр" и добавить поле "Должность" для фильтрации
3.3) Сохранить изменения фильтра

4: Форма редактирования должна отображаться на четырех закладках
4.1) Перейти в раздел редактирования любого элемента: Контент > Инфоблоки > Список элементов > Редактировать элемент
4.2) Настроить закладки формы редактирования следующим образом:

    Закладка 1:
        Поля: Активность, Начало активности, Название, Символьный код
        Свойства: Должность, Компания
    Закладка 2:
        Остальные используемые поля и свойства
    Закладка 3:
        SEO параметры (стандартная закладка)
    Закладка 4:
        Реклама (стандартная закладка)
        4.3. Для этого можно настроить вид отображения полей в форме редактирования через интерфейс системы управления

5: Изменение элементов должно фиксироваться для последующего анализа
5.1) Контент > Инфоблоки > Типы инфоблоков > Товары  услуги > Отзывы > Журнал событий >  Записывать изменение элемента
