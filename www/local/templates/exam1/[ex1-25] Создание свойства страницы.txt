Шаги для решения [ex1-25] Создание свойства страницы: управление цветом шапки левого меню

1.Добавить стили в файл стилей шаблона сайта 
    1.1 Перейти в администрирование сайта: Контент > Шаблоны > Шаблоны сайта > Выбрать текущий шаблон 
    1.2 Открыть файл стилей шаблона (style.css или аналогичный) для редактирования 
    1.3 В конец файла стилей добавить стили из задания
    1.4 Сохранить изменения

2. Создать пользовательское свойство страницы для управления цветом 
    2.1 Перейти в административный интерфейс: Настройки > Настройки продукта > Настройки модулей > Контент и структура 
    2.2 Найти раздел "Пользовательские поля" 
    2.3 Создать новое свойство для страниц: 
        - Название: Цвет шапки левого меню 
        - Код: MENU_HEADER_COLOR 
        - Тип: Список 
        - Значения: - red - green - Значение по умолчанию: red 
    2.4 Сохранить изменения

3. Вывести значение свойства страницы в шаблон сайта 
    3.1 Открыть файл шаблона, отвечающий за вывод левого меню (например, left_menu.php) 
    3.2 Перед выводом меню добавить код для проверки и использования значения свойства страницы
    3.3 Сохранить изменения

4. Установить значение по умолчанию для всего сайта 
    4.1 Перейти в административный раздел: Контент > Структура сайта 
    4.2 Открыть главную страницу сайта (/index.php) для редактирования 
    4.3 Перейти в настройки страницы: Свойства страницы 
    4.4 В поле "Цвет шапки левого меню" установить значение red (если оно не установлено автоматически) 
    4.5 Сохранить изменения

5. Задать значение green для страницы "История компании" 
    5.1 Перейти в раздел Контент > Структура сайта > Найти страницу /company/history.php 
    5.2 Открыть страницу для редактирования 
    5.3 Перейти в настройки страницы: Свойства страницы 
    5.4 В поле "Цвет шапки левого меню" установить значение green 
    5.5 Сохранить изменения
