1. Создать раздел "Отзывы" и добавить ссылку в главное меню

1.1. Создание раздела "Отзывы"

    Перейдите в административную панель сайта: Администрирование > Контент > Структура сайта.
    В списке разделов выберите пункт Создать раздел.
    В поле Название введите Отзывы.
    В поле Код папки укажите rew (это будет частью URL: /rew/).
    Нажмите Сохранить для создания нового раздела.

1.2. Добавление ссылки на раздел в главное меню

    Перейдите в административной панели: Администрирование > Контент > Меню.
    Выберите пункт Главное меню (или соответствующий тип меню, используемый на вашем сайте).
    Нажмите Добавить пункт меню.
    В поле Название пункта введите Отзывы.
    В поле Ссылка укажите путь к разделу: /rew/.
    Нажмите Сохранить для добавления новой ссылки в главное меню.

2. Настроить вывод списка отзывов и детальный просмотр с помощью комплексного компонента

2.1. Добавление комплексного компонента на страницу

    Перейдите в раздел /rew/, который вы создали на шаге 1.
    В этом разделе нажмите Редактировать раздел.
    Найдите блок для добавления компонента.
    Выберите тип компонента: Инфоблоки > Список и детальная страница (это комплексный компонент).
    Установите компонент на страницу.

2.2. Настройки компонента для отображения отзывов

    В настройках компонента укажите информационный блок Отзывы (тот, который был импортирован ранее из XML).
    В настройках компонента убедитесь, что включен режим ЧПУ для работы с символьными кодами элементов (это позволит формировать красивые ссылки для детальных страниц).
    Установите отображение 2 отзыва на странице для списка отзывов (этот параметр находится в блоке "Постраничная навигация").

3. Внедрить верстку для списка и детального просмотра отзывов

3.1. Загрузка файлов верстки

    Откройте административную панель: Администрирование > Настройки > Файловый менеджер.
    Перейдите в папку раздела /rew/, созданного на шаге 1.
    Загрузите файлы reviews-list.html и reviews-detail.html в эту папку (эти файлы предоставлены в материалах).
    После загрузки убедитесь, что файлы корректно отображаются в файловом менеджере.

3.2. Настройка отображения списка отзывов

    Перейдите в настройки комплексного компонента (найденного на шаге 2.1).
    В настройках компонента выберите шаблон компонента для списка элементов, используя файл reviews-list.html.
    В списке отзывов должны отображаться следующие данные:
        Фото (Картинка для анонса)
        Имя-фамилия (Название)
        Дата отзыва (Начало активности)
        Короткий текст (Описание для анонса)
        Компания
        Должность

3.3. Настройка отображения детального просмотра

    В настройках компонента для детальной страницы выберите шаблон с использованием файла reviews-detail.html.
    На детальной странице отзыва должны выводиться следующие данные:
        Фото (Детальная картинка)
        Полный текст отзыва (Детальное описание)
        Имя-фамилия (Название)
        Дата отзыва (Начало активности)
        Компания
        Должность
        Документы: если файлы не заданы для элемента, этот блок не отображается.

4. Настроить вывод изображений и заглушек

4.1. Настройка вывода изображений

    Если у элемента инфоблока отзывов отсутствует изображение, должно выводиться изображение-заглушка no_photo.jpg.
    Загрузите файл no_photo.jpg в файловый менеджер: Администрирование > Настройки > Файловый менеджер в папку /rew/ или в папку с изображениями сайта.
    Перейдите в настройки компонента и укажите путь к этому файлу для отображения в случае отсутствия изображения у отзыва.

4.2. Создание тестового отзыва без изображения

    Перейдите в административной панели: Контент > Инфоблоки > Отзывы > Добавить элемент.
    Заполните поля отзыва, но не загружайте изображение.
    Сохраните элемент и убедитесь, что на странице списка и детальной странице отображается заглушка no_photo.jpg.

5. Настроить постраничную навигацию

5.1. Включение постраничной навигации

    В настройках комплексного компонента включите постраничную навигацию (раздел "Настройки постраничной навигации").
    Укажите отображение по 2 отзыва на одной странице.
    Убедитесь, что название категорий для навигации соответствует отображаемому контенту, например: Отзывы 1-2, Отзывы 3-4.

6. Реализовать поддержку Эрмитажа для редактирования

6.1. Включение Эрмитажа

    В настройках комплексного компонента убедитесь, что активирована поддержка Эрмитажа.
    Это позволит редактировать элементы инфоблока прямо на странице списка отзывов.
    Откройте публичную часть сайта и убедитесь, что при наведении на элементы в списке отзывов появляется значок для редактирования.

7. Настроить режим работы ЧПУ

7.1. Включение режима ЧПУ для инфоблока

    Перейдите в административной панели: Контент > Инфоблоки > Типы инфоблоков > Отзывы.
    В настройках инфоблока включите режим работы ЧПУ для генерации символьных кодов элементов.
    Убедитесь, что новые отзывы, которые добавляются в инфоблок, открываются корректно в публичной части сайта по символьным кодам.

8. Задать метаданные для детальной страницы отзыва

8.1. Настройка шаблонов метаданных

    Перейдите в настройки комплексного компонента для детальной страницы отзывов.
    Укажите шаблоны для следующих метаданных:
        Заголовок окна браузера: Отзыв – [Имя Фамилия]
        Заголовок страницы (H1): Отзыв – [Имя Фамилия] – [Компания]
        Ключевые слова: лучшие, отзывы, [Компания]
        Описание: [Текст анонса]
    Убедитесь, что данные метаданные корректно подставляются для каждого элемента отзыва, основываясь на его полях (Имя, Фамилия, Компания, Текст анонса).
