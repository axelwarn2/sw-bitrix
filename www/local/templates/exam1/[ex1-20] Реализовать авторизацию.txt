Шаги для решения [ex1-20] Реализовать авторизацию: типовые сценарии, соцсети и captcha

1. Создать страницу для просмотра и редактирования профиля пользователя 
    1.1 Перейти в раздел администрирования: Контент > Структура сайта > Создать страницу 
    1.2 Указать: 
        - Путь: /login/user.php 
        - Название: Редактирование профиля 
    1.3 На созданной странице разместить компонент для редактирования профиля: 
        - Редактировать страницу > Добавить компонент > Пользователи > Форма редактирования профиля (system.auth.profile)

2. Внедрить компонент авторизации в шаблон сайта 
    2.1 Открыть шаблон сайта для редактирования: Администрирование > Настройки > Шаблоны > Выбрать используемый шаблон 
    2.2 Найти блок для авторизации в шаблоне (например, в шапке сайта) 
    2.3 Разместить компонент авторизации: 
            - Добавить компонент > Авторизация > system.auth.form 
    2.4 Использовать шаблон компонента авторизации из предоставленных материалов: 
            - Путь к шаблону: components/system.auth.form/demo

3. Добавить отображение информации для авторизованного пользователя 
    3.1 В шаблоне компонента авторизации изменить верстку, используя файл main_auth.html 
    3.2 В шаблоне предусмотреть отображение информации: 
        - Имя пользователя, если он авторизован 
        - Ссылки на страницу профиля и выход из системы 
    3.3 Проверить отображение верстки для авторизованных и неавторизованных пользователей

4. Реализовать 5 типовых сценариев авторизации 
    4.1 В шаблоне компонента авторизации добавить ссылки для: 
        - Авторизации (по умолчанию) 
        - Перехода на страницу регистрации: href="?register=yes" 
        - Перехода на страницу восстановления пароля: href="?forgot_password=yes" 
        - Перехода к профилю пользователя: href="/login/user.php" 
        - Выхода: href="?logout=yes" 
    4.2 Проверить работу всех сценариев на сайте: 
        - Авторизация - Выход из системы 
        - Переход на страницу регистрации 
        - Переход на страницу восстановления пароля 
        - Переход к странице редактирования профиля

5. Добавить поддержку авторизации через соцсети 
    5.1 В разделе Администрирование > Настройки > Настройки модулей > Социальные сервисы настроить подключение соцсетей (Facebook, Google, и т.д.) 
    5.2 Открыть шаблон компонента авторизации и добавить поддержку авторизации через соцсети: 
        - Разместить соответствующие иконки соцсетей в шаблоне 
    5.3 Убедиться, что ссылки на авторизацию через соцсети корректны и работают

6. Добавить поддержку captcha в шаблон компонента авторизации 
    6.1 Открыть шаблон компонента system.auth.form для редактирования 
    6.2 Вставить код для отображения captcha в форме авторизации: 
        - Использовать стандартный механизм captcha, поддерживаемый Bitrix 
    6.3 Настроить параметры captcha: 
        - Установить включение captcha после 1 неправильного ввода пароля 
        - Настроить для группы "Все пользователи" (включая неавторизованных) 
    6.4 Проверить работу captcha: 
        - Сделать неправильный ввод пароля и убедиться, что captcha появляется
